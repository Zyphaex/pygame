# Pygame Playground

---

## Requirements

Make sure you have Python and pip installed:

```bash
python --version
# Python 3.11.4
```
```bash
pip --version
# pip 23.2.1
```

## Installation

Install the required package and run the game:

```bash
pip install pygame-ce
```
```bash
python main.py
```
